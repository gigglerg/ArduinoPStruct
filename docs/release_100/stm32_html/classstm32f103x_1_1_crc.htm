Detailed Description

A class offering h/w CRC module access via API for device STM32F103C8T6.

Note
Class name intentionally generic
Attention
Ether use OpenSTM32 HAL or for Arduino install library Arduino_Core_STM32 https://github.com/stm32duino/Arduino_Core_STM32/
HAL_CRC_MODULE_ENABLED macro must be defined to use this class
