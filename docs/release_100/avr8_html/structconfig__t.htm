Detailed Description

Application specific ADT, you define this to suit your data requirements.

it is written to and read from persistent storage and you will access your data via this type. Most likely this will be a configuration structure of some form.

Note
packing not important whatever gives best optimisation either for access by cpu (typically padded) or storage (not padded)
